# Modelado de Hospital

Modelo completo para un sistema de gestión hospitalaria.

## Componentes del Sistema

- Pacientes
- Médicos
- Citas
- Historiales Médicos
- Departamentos
- Medicinas

## Objetivo

Proporcionar una base de datos robusta para la administración de un hospital moderno.
