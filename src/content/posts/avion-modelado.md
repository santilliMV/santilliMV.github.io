# Modelado de Avión

Este es un ejemplo de modelado entidad-relación para un sistema de gestión de aviones.

## Componentes Principales

- Aeronave
- Rutas
- Pilotos
- Itinerarios

## Características

El modelo permite gestionar la información completa de una aeronave y sus operaciones.
